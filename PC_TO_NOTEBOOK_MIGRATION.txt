========================================
   PC TO NOTEBOOK MIGRATION GUIDE
========================================

PREREQUISITES NEEDED:
- Python 3.9+ 
- Node.js & npm
- MongoDB (local install OR cloud Atlas account)
- Git
- Claude API key (Anthropic)

ESTIMATED TIME: 2-3 hours

========================================
STEP 1: CLONE & SETUP BACKEND
========================================

1. Clone the repository
   git clone [your-repo-url]
   cd upload

2. Create virtual environment
   cd backend
   python -m venv venv_new
   
   # Windows:
   venv_new\Scripts\activate
   
   # Mac/Linux:
   source venv_new/bin/activate

3. Install Python dependencies
   pip install -r requirements.txt
   playwright install

4. Create .env file in backend folder:
   ANTHROPIC_API_KEY=your_claude_api_key_here
   MONGODB_URI=mongodb://localhost:27017/
   MONGO_DB_NAME=ui_templates

========================================
STEP 2: SETUP FRONTEND
========================================

1. Install Node dependencies
   cd frontend
   npm install

========================================
STEP 3: DATABASE SETUP
========================================

Option A - Local MongoDB:
- Install MongoDB Community Server
- Start MongoDB service
- Database will auto-create on first run

Option B - Cloud MongoDB:
- Create MongoDB Atlas account
- Get connection string
- Update MONGODB_URI in .env file

========================================
STEP 4: START SYSTEM
========================================

Option A - Use batch file (Windows):
   start_system.bat

Option B - Manual start:
   1. Backend: cd backend && python main.py
   2. Frontend: cd frontend && npm start

Access points:
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API Docs: http://localhost:8000/docs

========================================
POTENTIAL ISSUES & SOLUTIONS
========================================

Issue: Playwright browser download fails
Solution: Run "playwright install" manually

Issue: MongoDB connection fails
Solution: Check MongoDB service is running or verify Atlas connection string

Issue: Python packages fail to install
Solution: Ensure you're in the virtual environment and have Python 3.9+

Issue: Frontend won't start
Solution: Delete node_modules folder and run "npm install" again

Issue: CORS errors
Solution: Verify backend is running on port 8000

========================================
VERIFICATION CHECKLIST
========================================

□ Python virtual environment created and activated
□ All pip packages installed successfully
□ Playwright browsers downloaded
□ .env file created with API key
□ MongoDB connection working
□ Node packages installed

========================================
FILES TO BACKUP/TRANSFER
========================================

Essential:
- Source code (entire project folder)
- .env file (contains API keys)

Optional but recommended:
- Any custom UI templates in UIpages/
- Database export if you have important data

========================================
NOTES
========================================

- The project is designed to be cross-platform
- Virtual environment isolates Python dependencies
- MongoDB data will need to be re-seeded on new machine
- Temp files (screenshots, UI codes) can be ignored
- Make sure ports 3000, 8000 are available
